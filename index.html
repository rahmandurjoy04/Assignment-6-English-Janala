Frequently Asked Questions

1.what is the difference between var, let, and const?

Ans:

var : var is a function scoped variable, that means it is accessible throughout the function. Var Can be reassigned and updated.

let : let is a block scoped variable, that means it is accessible only within the block, say { }. let Can also be reassigned and updated.

const : const is a block scoped variable, that means it is accessible only within the block, say { }. const Can neither be reassigned nor updated.

What are the difference between map(), forEach(), and filter()?

Ans:

Map() : Map() creates a new array by applying a function to each element of the original array. We use map() when we need to transform data (apply any operation to data) and also want to store them

forEach() : forEach() executes a function on each element of the array but doesn't return an array. We mat use it in the purpose of validation, which doesnot require a new array.

filter() : filter() is used when we want to apply certain conditions on array elements. It returns a new array with the elements that satisfy the given condition.

Explain arrow functions and how they are different from regular functions

Ans:

Arrow function is nothing but a regular function with a shorter form. It starts with a declaration, then parameters and after that an arrow. The code after the array will execute.
The form of an array function may be:
variable_type   variable_name   = (parameters) => {
//Function Body
}
It may have a parameter or may not.

Discuss how JavaScript Promises work?

Ans:

Promises are useful for asynchronus operations like fetching data from a server or reading files.
A promise has 3 states. They are:

Pending :This is the initialstate, the operation is still ongoing.

Fullfilled : The operation completed successfully, and the Promise has a resulting value.

Rejected : The operation failed, and the Promise has a reason (error)..

Explain how closures work in JavaScript?

Ans:

A closure happens when ew have an inner function defined inside an outer function. The inner function has access to its own variables ,the variables of the outer functions and also the global variables. Closure can be used to create reusable functions.
